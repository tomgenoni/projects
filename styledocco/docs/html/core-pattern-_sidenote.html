Sidenote

Basic styling for a sidenote, can be .width-half.

By using <h4> and <h5> this will not show up in Outline View. Use a <section> with an <h2> & <h3> if you want the content to appear in Outline View. 

.sidenote {
  background: $color-4;
  padding: 20px;
  position: relative;

  h4 {
    @include sans-med;
    font-size: 1.3em;
    margin-bottom: .2em;
    color: $color-1;
  }

  h5 {
    @include sans;
    font-size: .9em;
    text-transform: uppercase;
    margin-bottom: .4em;
  }

  p, ul li {
    @include sans;
  }
}

.sidenote-icon {
  @extend .sidenote;
  background-image: url(../img/icon.png);
  background-repeat: no-repeat;
  background-position: 20px 20px;
  @include background-size(55px);

  h4, h5 {
    padding-left: 70px;
  }

  &.width-half {
    @include width-half;
  }
}
