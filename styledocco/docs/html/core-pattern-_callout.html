Callout

This provides all code required to make default callouts on Inkling.

¶

Mixin: Callout Heading

This mixin creates the bar across the top of callouts that have one. It uses the @include image-mask mixin for the icon. See the Basic and Audio/Video callouts for examples. 

@mixin callout-heading($image) {
    h4 {
        padding-left: 50px;
        &:before {
            content: "";
            @include image-mask(30px,#{$image},30px,30px,#fff);
            position: absolute;
            top: 5px;
            left: 10px
        }
    }
}

Mixin: Arrow Right

Creates the dark background and arrow that appears to the right on nearly all the callouts. 

@mixin arrow-right($bgcolor) {
    &:after {
        content: "";
        display: block;
        position: absolute;
        top: 3px;
        bottom: 3px;
        right:3px;
        width: 30px;
        @include border-radius(3px);
        background: $bgcolor url(ui/callout-arrow.png) no-repeat 50% 50%;
        @include background-size(6px 14px);
    }
}

Callout: Shared Code

This holds the code for all the callout types.

Note the optional .stack class can only be used when there is no top bar â€” that is, an <h4> in the code. 

.callout {
    background: $color-4;
    @include border-radius(3px);

    padding: 8px 15px 15px 15px;

        background: $color-1;

        padding: 8px 30px 8px 13px;
        @include sans-med(1.1em);
        position: relative;
        z-index: 1;
        @include arrow-right(transparent);

        // added to make the right arrow line up with the h5 arrows
        // overrides width in '@include arrow-right'
        &:after {
            width: 24px;
        }

        @media screen and (max-width: 480px) { 
            font-size: 1.0em;
        }
   }

    figure {
        width: 145px;
        float: right;
        margin: 0 0 0 1em;

        img {
            margin-bottom: 0;
        }

        @media screen and (max-width: 480px) { 
            float: none;
            width: auto;  
            margin: -15px 0 .7em 0; 
        }
    }

    h5 {
        @include sans-med(1.4em,1em);
        margin-bottom: .2em;        
        color: $color-1;

        &[href] {
            margin: 0 0 5px;
            border-bottom: 1px dashed #ccc;
            padding: 12px 15px 12px 0;
            position: relative;

            &:last-child {
                border: none;
                margin-bottom: 0;
            }
        }    

        @media screen and (max-width: 480px) { 
            padding-left: 0;
            font-size: 1.2em;
        }
    }

    h4 + h5[href] {

    }

    h6 {

        color: $color-1;
    }

    p {

    }

    &.width-half {

            width: auto;
            margin: -15px 0 .7em 0; 
        }
    }
    &.stack {
        // 1px is required to allow the margin of the h4

        h4 {
            background-image: none;
            margin: 2px 3px 10px;
            &:after {
                background-image: none;
            }
        }

            padding: 15px 15px 15px 55px;
            margin-bottom: 0;
            @include arrow-right($color-1);

            &:before {
                content: "";
                background: $color-1;
                position: absolute;
                left: 15px;
                top: 50%;
                margin-top: -16px;
                z-index: 1;
                width: 30px;
                height: 32px;
            }
        }
    }

Callout: Types

These are classes added to the default .callout class to provide specific icons for the top bar.

Note that .callout-custom uses differnt CSS for the <h4> as any custom image is likely not to be used as an image-mask. In this case we're putting the image as a background-image on the <h4>. 

.callout-web {
    @include callout-heading('callout-web');
}
.callout-audio {
    @include callout-heading('callout-audio');
}
.callout-video {
    @include callout-heading('callout-video');
}
.callout-custom {
    @include callout-heading('callout-custom');
    h4 {
        padding-left: 50px;
        background-image: url("ui/callout-custom.png");
        background-repeat: no-repeat;
        background-position: 10px 5px;
        @include background-size(30px,30px);
        &:before {
            content: none;
        }
    }
}

Callout: Simple

The simplest style of callout, does not have a top bar.

Note that this does not support images. 

.callout-simple {
    @include display-box;
    @include box-orient(vertical);
    @include box-pack(center);
    position: relative;
    min-height: 74px;
    padding-left: 0;

    @media screen and (max-width: 480px) { 
        min-height: 50px;
        padding-bottom: 6px;
    }

    h5 {
        margin-bottom: 0;
        padding: 0 60px 0 15px;
        @include arrow-right($color-1);
    }
    p {
        padding: 5px 60px 0 15px;
    }

    &.stack {
        display: block;
    }
}

Callout: Assessment

Simple style callout with asssement type icon to the left.

  • @extend .callout-simple; extends the code for Callout: Simple.
  • &:before adds the image via the @include image-mask;.

Options are:

  • .callout-assessment
  • .callout-key-term
  • .callout-table
.callout-assessment {
    @extend .callout-simple;

    h5 {
        padding-left: 70px;
        @include arrow-right($color-1);

        @media screen and (max-width: 480px) { 
           padding-left: 55px;
        }

        &:before {
            content: "";
            position: absolute;
            left: 15px;
            top: 50%;
            margin-top: -20px;
            z-index: 1;
            @include image-mask(100%,assessment,41px,45px,$color-1);

            @media screen and (max-width: 480px) { 
               width: 30px;
               height: 34px;
               margin-top: -14px;
           }
        }
    }
}

.callout-key-term {
    @extend .callout-assessment;
    
    h5:before {
        -webkit-mask-image: url('ui/keyterms.png');
   }
}
.callout-table {
    @extend .callout-assessment;
    
    h5:before {
        -webkit-mask-image: url('ui/table.png');
    }
}

Based on the Basic style but allows for multiple links. Note that HTML is different when using multiple links. See 3D and Molecule in the Pattern Library for HTML. 

.callout-threed {
    @include callout-heading('callout-custom');
    padding-bottom: 5px;

    h4:before {
        -webkit-mask-image: url("ui/threed.png");
        -webkit-mask-size: 27px;
    }

    &.stack {
        h4 {
            padding-left: 15px;
            &:before {
                content: none;
            }
        }
        h5 {
            &:before {
                @include image-mask(100%,threed,30px,34px,$color-1);
            }
        }
    }
}

.callout-molecule {
    @include callout-heading('callout-custom');

Callout: Transcript

Often used with Inline Audio callout.

.callout + .callout-transcript is used to remove counteract the default margin that would otherwise push it too far down. 

.callout-transcript {
    position: relative;
    min-height: 0;
    padding-bottom: 12px;

    h5 {
        @include arrow-right($color-1);
        margin-bottom: 0;
        font-size: 1.1em;
    }
}

.callout + .callout-transcript {
    margin-top: -.7em;
}
