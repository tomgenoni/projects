Text Images

Used for tables, charts and equations. Anything that has text that needs to be readable by the end user and should not grow larger than it needs to.

All text images will have a class name entered into /assets/css/text-image.css

This is default behavior for all text images (except for equations). 

.text-image {
  max-width: 100%;
  margin: 0 auto;
}
¶

Block Image Equations

figure > img.equation,
.equation-key > img.equation {
  margin-left: auto;
  margin-right: auto;
  height: auto;
  padding: 0 10px;
}
¶

Inline Image Equations

p > img.equation,
li > img.equation,
figcaption img.equation {
  height: auto;
  display: inline-block;
  vertical-align: -.1em;
}
¶

HTML Equations

Both inline and block. 

div.equation {
  text-align: center;
  margin: 1em auto;
  display: inline-block;
  padding: 0 10px;
  white-space: nowrap;
}
span.equation {
  white-space: nowrap;
}
