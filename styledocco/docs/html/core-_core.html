Core Configuration

This acts as configuration file, calling all the partials used by 'core'.

Compass Libraries

We rely on these for CSS3 and some typography mixins 

@import "compass/css3";
@import "compass/typography";

Common

Partials for setting default mixins, colors and fonts. 

@import "common/reset";
@import "common/mixin";
@import "common/color";
@import "common/layout";
@import "common/typography";
¶

Custom Color

Includes the color theme and varaibles added from 'custom'. This allows users to create color themes that will color the patterns and elements that use color variables. 

@import "../custom/common/color";
