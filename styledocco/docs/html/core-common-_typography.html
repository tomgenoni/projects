Typography

Mixins and classes related to using and declaring fonts.

¶

Font Mixins

These should be used at all times to declare fonts. All values are optional:

  • $font-size - If not declared any previous font-size declaration will apply.
  • $line-height - If not declared any previous line-height declaration will apply.
  • $text-transform - If not declared text-transform will not change.
  • $letter-spacing - If not declared letter-spacing will not change.

Note that font-weight changes depending on the mixin in called.

Do not use font-weight:bold or font-weight:normal. You must declare a font-weight between 400 and 700. 

@mixin sans($font-size: false, $line-height: false, $text-transform: none, $letter-spacing: inherit) {
    font-family: 'Whitney A', 'Whitney B', Whitney, "Verdana", Helvetica, sans;
    @if $font-size { font-size: $font-size; } 
    @if $line-height { line-height: $line-height; } 
    text-transform: $text-transform;
    letter-spacing: $letter-spacing;
    font-weight: 400;
}
@mixin sans-med($font-size: false, $line-height: false, $text-transform: none, $letter-spacing: inherit) {

Text Serif

Used commonly for math characters that do not render in Chronicle. See the following code example:

This β is an example of a beta symbol.

 <p>This <span class="text-serif">β</span> is an example of a beta symbol.</p>
.text-serif {
    font-family: 'Times New Roman', Times, serif;
}
¶

Force Break

Forces breaks in long strings, most often used in unbroken strings like http://url.com/that/goes/on/and/will/not/break/otherwise that appear in table-cells where break-word does not work. 

.force-break {
    word-break: break-all;
}

Helper Styles

.indent - Forces indent, most often used in paragraphs where indentation is desired.

Other classes should be self-explanatory. 

.indent {
    text-indent: 2em !important;
}
@media screen and (max-width: 480px) {
    .indent {
        text-indent: 1.5em !important;
    }
}

.no-indent {
    text-indent: 0 !important;
}

.no-bold {
    font-weight: 400 !important;
}

.no-italic {
    font-style: normal !important;
}

.underline {
    text-decoration: underline !important;
}

.no-line-break {
    white-space: nowrap !important;
}

.center-align {
    text-align: center !important;
}

.right-align {
    text-align: right !important;
}

.double-space {
    line-height: 2em !important;
}

.lowercase {
    text-transform: lowercase !important;
}

.uppercase {
    text-transform: uppercase !important;
}

.overline {
    text-decoration: overline !important;
}
