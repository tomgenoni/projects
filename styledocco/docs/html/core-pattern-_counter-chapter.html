Counter Chapter

Important: These counters cannot be read with screen readers for accessiblity reasons this method should be avoided. It is provided when hard-coded methods are not practical.

A counter is constructed based on the 'chapterXX' class that's on the body tag. In the example 'chapter09' is wrapping the code. You may need to create your own counter type if you have more specific needs. 

$number-of-chapters: 30;

@for $i from 1 through 9 {
  .chapter0#{$i} {counter-reset: counter-chapter #{$i} counter-sub-1 counter-sub-2}
}    
@for $i from 10 through $number-of-chapters {
  .chapter#{$i} {counter-reset: counter-chapter #{$i} counter-sub-1 counter-sub-2}
}

.counter-chapter {

	li {
		list-style: none;

		&:before {
			display: inline-block;
			margin: 0 0 0 -1.5em;
			counter-increment: counter-sub-1;
			content: counter(counter-chapter)"."counter(counter-sub-1);
			padding-right: 1em;
		}

		& > ol > li:before {
			content: counter(counter-chapter)"."counter(counter-sub-1)"."counter(counter-sub-2);
			counter-increment: counter-sub-2;
		}
	}
}
