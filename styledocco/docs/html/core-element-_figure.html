Figure

Default styling for all images, photographs, illustrations and maps.

.width-full - Figure will runs the width of the page.

.width-half - Floats the figure on the right. This is a global class available to many patterns.

Figcaption Truncation

  • 1 line when figure is .width-full
  • 3 lines when figure is .width-half

Note: Figcaption truncation can be overriden on a global basis, turning it off completely. See Figure Caption Truncation in the Pattern Library for an example.

The & + img at the bottom provies the spacing for when the figcaption are on top of the img. 

$figure-border-color: #59b0dd;

figure {
    position: relative;
    @include sans(1em);
    margin: 1em 0;
    clear: both;

    > img {
        margin-bottom: 1em;
        @include box-sizing(border-box); 
        width: 100%;
    }

    &.width-full {
        width: auto;
    }

    figcaption {
        font-weight: normal;
        line-height: 1.4em;
        
        .figure-number {
            font-weight: 700;
            margin-right: 0.3em;
            padding: 0.1em 1.4em 0.1em 0;
            background: url('ui/link.png') 100% 42% no-repeat;
            @include background-size(auto 1em);
            color: $platform-color;
        }
        .caption-title { 
            font-weight: 600;
        }

        & + img {
          margin-bottom: 1.5em;
          margin-top: .8em;  
        }
    }
}
