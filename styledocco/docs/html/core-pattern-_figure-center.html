Figure Caption Position

Centers figure and caption, maxing out at 70% width. 

figure {
	&.center {
		max-width: 70%;
		margin-right: auto;
		margin-left: auto;
		background: none;
		padding:0;	
	    @media screen and (max-width: 480px) { 
	        max-width: 100%;
	    }
	}
}
